# Free weight vs Machine Preacher curl

Free Weight vs Machine Preacher Curl. what is better for muscle growth?

We know to train the biceps we want train their function of elbow flexion. we also know that we want to train the biceps where it has best leverage in elbow flexion which is at the bottom of the rep (descending rep). does the variation matter whether it be machine or free weight?

free weights will be relatively stable and it has a good resistance profile which makes it a great exercise. though stability does decrease if trying to do a unilateral variation. but anyway it wont be as stable as an machine with a fixed arm path especially with some setups being standing.

for the machine it depends on the brand. some machines will be descending which is great but some might be ascending (hardest at the top) meaning it will be a brachioradialis movement. but it is very stable being seated with a fixed arm path whether you do it bilateral or unilateral.

so which one should you do? the machine will always be better if the machine has a good resistance profile and is comfortable for you. but if it has a bad resistance profile its fine to take away a little bit of that stability and do a free-weight preacher for a better resistance profile
