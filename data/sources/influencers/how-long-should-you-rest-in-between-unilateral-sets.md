# How long should you rest in between unilateral sets?

How long should you rest in between unilateral sets? Does resting increase muscle growth?

many people when doing unilateral sets go from one side to another back to back without rest but can this be detrimental? is there a benefit in resting when the muscle on the other side hasnt had any muscular related fatigue?

A "unilateral set" in the context of exercise refers to performing a set of repetitions for an exercise, focusing on one side of the body at a time.

you actually should rest at least 2-3 minutes between sides. this is because the first set will be causing supraspinal cns fatigue. this will affect the other side as without adequate rest it will result in less motor unit recruitment which means less performance in the following side.

Supraspinal CNS fatigue, a type of central fatigue, reduces the brain's ability to activate motor units, leading to decreased muscle activation during exercise.

the only time i would recommend going back to back is if youre trying to fix a muscle imbalance so you do weak side into strong side without rest
