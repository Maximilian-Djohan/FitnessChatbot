# how good is the TNF alternating leg extension?

How good is the TNF alternating leg extension for muscle growth?

we know that a single limb exercise will have more motor unit recruitment compared to a bilateral version of an exercise which will cause you to train more muscle fibers. this is as you focus central motor command to one limb it will lead to more motor unit recruitment as you wont have to divide it between limbs. so what does this mean?

The bilateral force deficit is a phenomenon where the total force produced when using both limbs at the same time is less than the sum of the forces each limb could produce individually.

but what if we instead compared this to just performing it one limb rest into another limb instead of doing alternating reps. well as the set will be prolonged you will suffer more from central nervous system fatigue. this longer set time will be worse than a standard unilateral set as the prolonged set time will hinder the brains ability to recruit as the set goes on.

so what should you do? just do bilateral if you can (stability from other leg as counter force will increase MUR). if you max it out then switch to unilateral. if unilateral takes too long or if you have trouble stabilizing you can experiment with doing this alternating leg variation.
