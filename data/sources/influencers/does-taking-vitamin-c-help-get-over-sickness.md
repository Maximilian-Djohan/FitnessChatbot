# does taking vitamin c help get over sickness?

Does taking vitamin C actually help get over sickness?

people often talk about taking vitamin c to help combat and alleviate sickness but does this have any scientific background supporting this or is it just a myth?

Yes there is a benefit of taking vitamin c. taking 1g of vitamin c once a day or 1g twice a day is shown to reduce the severity and duration of your sickness.

(though as a preventative measure it hasnt been shown to have a significant effect)

Vitamin C reduces the severity of common colds: a meta-analysis
Harri Hemila, Elizabeth Chalker
PMID: 38082300 PMCID: PMC10712193 DOI: 10.1186/s12889-023-17229-8
