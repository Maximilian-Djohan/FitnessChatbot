# are these exercises the same?

Are these exercises are actually the same?

all of these exercises look very different visually with how they are performed but they are actually very similar. but how is this? in all of these different exercises you are actually performing the same action being hip extension. so what does this mean?

hip extension through these exercises will train the hip extensors: the adductor magnus, glutes and hamstrings. so if these exercises all perform the same function whats the purpose of doing them differently?

through different exercises you can bias different hip extensors with the setup. through resistance profile/range of the motion, or bending the knee to create antagonistic inhibition in the hamstrings or to keep it straight to work the hamstrings. there are different reasons to perform these hip extensions challenges.
