# What is the best incline for chest growth?

What is the best bench incline for upper pec growth?

Jeff Nippard tested upper pec activation at every bench incline from flat to upright. the results showed that there was decent activation from 0-65 degrees which the best being around 25-45 degrees but why is this?

well starting off with the flat bench it might surprise being higher than you think it would be around 59% Maximum Voluntary Contraction. but this makes sense as you are performing horizontal adduction and while this will bias the mid pec all parts of the chest will be contributing and see growth.

but we can see it peaks towards that 45 degree. what factors make this happen? first is the resistance with the incline making the hardest part of the exercise a bit higher up in the rep in the mid range which is better for the upper pec. but more importantly it also makes his arm path a bit more tucked which we know is the most important factor for the upper as its primary function is shoulder flexion which happens when you're more tucked.

so what bench incline should you use? we know that arm path is the priority so before worrying about that we should make sure to use an arm path that allows us to primarily perform shoulder flexion. but after that we can use a 30-45 degree incline which would be best for the upper pec.
