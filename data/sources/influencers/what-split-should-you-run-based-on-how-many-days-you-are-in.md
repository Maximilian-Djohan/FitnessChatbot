# What split should you run based on how many days you are in the gym?

What split should you run based on how many days you can go to the gym?

we know that you want to be as consistent as possible and train every muscle group as frequently possible and a good way to do this is through 2 day splits like UL or a 3 day split like FB. but what split should you run if you are limited on what days you can workout?

well the two obvious choices we already went over are UL and FB as you will have 3/4 days of rest, and for fb you can do it in a way that fits you MWF, TTHS etc. the same concept goes for upper lower where you can shift around the days. and if your session is running too long use compounds to cut down on exercises

but what if you can only work twice in one week? to progress youll still want to maintain 2x frequency so you should run FB 2x with more volume then you would run on 3x. and if you only have 2 consecutive days you should still run FB but with first day having lower volume then the next day having more volume.

what if you can only workout 2 consecutive days like on the weekend but also have one day during the week? and you can run FB one day during the week and UL on the weekend.

and if you have 5-6 days any of the other mentioned splits work. and if you really enjoy being in the gym as much as possible then you can run something like upper lower 6x as long as you manage your volume.

so if you are limited on the days you can workout you should still maintain at least 2x frequency as we can see that even 1 set twice a week is enough to cause good growth in trained lifters.
